# INVESTMENT 📈

**其他语言版本: [English](README.md), [中文](README_zh.md).**

一个基于 Java 的命令行工具：从 Yahoo Finance 拉取日线行情，计算 **200 日均线（SMA200）** 与 **乖离率（Bias）**，并按偏离区间输出 **1x-5x** 金字塔定投建议（JDK 21 / Maven）。

## ✨ 功能亮点

- 🧠 **规则驱动**：按乖离区间执行，不受情绪干扰。
- 📊 **自动计算**：拉取日线、计算 SMA200、输出最新乖离率与区间。
- 🧱 **可扩展架构**：通过 `MarketDataProvider` 接口可替换行情源。
- 🧪 **测试覆盖**：核心档位边界与均线逻辑均有单元测试。

## 🚀 快速开始

### 1) 环境要求

- Java 21
- Maven 3.6+
- 可访问外网（Yahoo Finance）

### 2) 构建与测试

```bash
mvn clean test
```

### 3) 运行示例

```bash
mvn -q exec:java "-Dexec.args=--symbol QQQ --range 2y --base-amount 100"
```

## ⚙️ CLI 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--symbol` | 标的代码（如 `QQQ`、`^IXIC`） | `QQQ` |
| `--range` | Yahoo Chart API 区间，需覆盖至少 200 个交易日 | `2y` |
| `--base-amount` | 可选；基础定投金额，用于换算今日建议投入 | 无 |

## 🧮 策略规则（基于乖离率的金字塔）

乖离率公式：

`(当日收盘 / SMA200 - 1) * 100%`

| 乖离率区间 | 建议倍数 | 区间说明 |
|-----------|----------|----------|
| `>= 0%` | `1x` | 水上 / 基础定投 |
| `[-5%, 0)` | `2x` | 浅度回调 |
| `[-10%, -5%)` | `3x` | 常规回调 |
| `[-20%, -10%)` | `4x` | 深度回调（熊市加仓档） |
| `< -20%` | `5x` | 极度超卖 |

## 🧭 投资纪律

- 严格按规则执行：根据乖离区间决策，不靠情绪加减仓。
- 保持基础定投金额稳定，仅按模型倍数（1x-5x）进行放大。
- 在高倍加仓阶段前，确保有充足的场外现金储备。
- 坚持长期一致性，不在回撤期做随意的人为改动。
- 本项目仅用于纪律辅助，不构成投资建议。
